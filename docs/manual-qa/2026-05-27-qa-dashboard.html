Before you start

  1. Open the live admin: d3h8i7y9p8lyw7.cloudfront.net/admin/ and log in.
  2. Hard-refresh once (Ctrl/Cmd+Shift+R) so you get today's deploy, not a cached build.
  3. Language toggle is top-right (English / עברית) — several cards ask you to check both.
  4. To make a "new shelf" for the #57 tests: Map Files → Download a floor (e.g. floor_0.svg), open it in a text editor, add one rect inside the <svg>: 
    <rect id="qa-new-shelf-1" data-map-object="shelf"
      x="60" y="60" width="80" height="40" fill="#cccccc"/>
    save, then Replace that floor with your edited file. (A "strict superset" upload — nothing removed — so it won't trigger the removed-shelf wizard, only the new-shelf review.)
  5. Mark each card Pass / Fail / Blocked and jot notes (saved automatically — stop and resume anytime).
  6. How to send me results — there is no live "ping" in this offline dashboard. When you're done (or stuck on a card), click 📋 Copy results for Claude at the top and paste it into our chat. Or just tell me in chat which card + what you saw (exactly like you did for C1). (If you'd rather ping me per-card in real time, ask me to "turn on the live bridge" and I'll switch the dashboard to its interactive mode.)
